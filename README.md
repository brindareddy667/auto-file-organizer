# Auto-File-Organizer 📂
A Python-based productivity tool to keep your directories clean.

## How it Works
The script scans a directory and automatically moves files into subfolders based on their extensions:
- **Images:** .jpg, .png, etc.
- **Documents:** .pdf, .docx, etc.
- **Archives:** .zip, .rar, etc.

## Usage
1. Place `organizer.py` in the cluttered folder.
2. Run `python organizer.py`.
3. Watch your files disappear into neat categories!
